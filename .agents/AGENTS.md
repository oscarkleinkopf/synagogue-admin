# Reglas de KehiláAdmin

- Este proyecto es una Single Page Application (SPA) estática.
- No utilizamos entornos de Node.js, npm run o frameworks complejos (React/Next) a menos que se pida explícitamente.
- Todo se construye con HTML5, CSS vanilla (usando variables personalizadas y HSL) y JavaScript puro (Vanilla JS).
- El almacenamiento se gestiona exclusivamente a través de `localStorage` mediante la clase `DataStore`.
